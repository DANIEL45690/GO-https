## 🎯 Описание

**Go-Style HTTP Toolkit** — мощный консольный HTTP-клиент в стиле Go, объединяющий функциональность Postman, curl и wget в одном инструменте. Градиентный интерфейс, анимации и 50+ функций для работы с HTTP.

> Идеален для разработчиков API, тестировщиков, пентестеров и DevOps инженеров.
